## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic understanding of HTML/CSS/JavaScript (for customization)

### Installation & Setup

1. **Clone the repository**
   \`\`\`bash
   git clone https://github.com/not-amarnath/portfolio.git
   cd portfolio
   \`\`\`

2. **Open locally**
   \`\`\`bash
   # Option 1: Open directly in browser
   open index.html
   
   # Option 2: Use a local server (recommended)
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (if you have live-server installed)
   npx live-server
   \`\`\`

3. **View the portfolio**
   - Direct: Open `index.html` in your browser
   - Local server: Navigate to `http://localhost:8000`

### Customization

1. **Update Personal Information**
   - Edit `index.html` to update name, bio, and contact details
   - Replace `anime_img_updated.jpg` with your profile picture
   - Update social media links in both navigation and contact sections

2. **Modify Projects**
   - Update project information in the Projects section
   - Replace project images in the `project-img` divs
   - Update GitHub and demo links

3. **Customize Styling**
   - Edit `style.css` to change colors, fonts, or layout
   - Current theme uses `#077b32` as the primary accent color
   - Modify the `@import` statement to change fonts

4. **Update Resume**
   - Replace `resume_updated.pdf` with your latest resume
   - Update the resume link in the "View Resume" button
