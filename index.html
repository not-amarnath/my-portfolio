Energy-IoT

React Next.js Tailwind CSS

A smart platform that enables households to track and control their electricity consumption in real time. Using IoT-enabled devices with sensors and ESP32/ESP8266 controllers, the system captures live energy data and syncs it to a cloud backend powered by Node.js/Flask and Firebase. Users can access a web dashboard to monitor usage, analyze trends, and remotely switch appliances on or off. By providing insights, smart alerts, and remote control, the system promotes energy efficiency, reduces wastage, and supports sustainable living through intelligent energy management.
